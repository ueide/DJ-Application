Dual-Deck DJ Application

High-performance desktop application built with Modern C++ and JUCE framework.

  • Real-Time Low-Latency DSP: Implements a custom 3-band frequency isolation filter and logarithmic crossfader curves.
  • 60 FPS Custom UI Rendering: Bypasses default OS window components to manually paint high-fidelity vector graphics (SVGs).
  • Multi-Threaded Architecture: Runs background audio file rendering completely separate from the UI thread to eliminate lag.
  • State & Playlist Persistence: Automatically serializes and safe-loads user configuration changes locally via JSON.
